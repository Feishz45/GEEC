# Interface Gráfica Arduino - IGA

Projeto voltado à criação de uma interface gráfica para controle e monitoramento de um sistema Arduino utilizado em experimentos no laboratório de Engenharia Química.

## Objetivo
Desenvolver uma aplicação desktop em C# que permita a comunicação com placas Arduino via porta serial, exibição de dados em tempo real e envio de comandos.

## Tecnologia
- **Linguagem:** C#
- **Framework:** .NET Windows Forms
- **Hardware:** Arduino Uno
- **Comunicação:** Serial (porta COM)

## Funcionalidades
- Comunicação serial com o Arduino
- Exibição de dados em tempo real (sensores)
- Controles de envio (botões, sliders, campos)
- Interface amigável com foco na usabilidade
