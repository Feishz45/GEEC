## Objetivos do Grupo
- Aplicação prática dos conteúdos aprendidos em sala.
- Promover a troca de conhecimentos entre os membros.
- Desenvolver projetos reais e úteis para a comunidade acadêmica.
- Explorar tecnologias emergentes.
